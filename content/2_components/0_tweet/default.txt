Title: Tweet

----

Content:

Embeds a tweet and asynchronously loads the required Twitter library to do so.

(example: tweet active: result)

**Note:** The HTML element that is the Oblik component must be a `<blockquote>` as per the [Twitter documentation](https://developer.twitter.com/en/docs/twitter-for-websites/embedded-tweets/overview).

## Options

### `url`

The tweet URL. Looks like `https://twitter.com/{{ USER }}/status/{{ ID }}`

**Required.**

### `lang`

In what language the tweet's UI is displayed.

**Default:** `en`
